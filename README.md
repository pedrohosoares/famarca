## Famarca teste

Repositório sobre api para farmaca

## Endpoints

Ver todos colaboradores, ver colaborador especifico, cadastrar colaborador, atualizar colaborador, deletar colaborador
- GET /api/colaboradores
- GET /api/colaborador/{id}
- POST /api/colaborador
- PUT /api/colaborador/{id}
- DELETE /api/colaborador/{id}

- POST /api/salario/
- DELETE /api/salario/{id}


## Instalação

PHP Version 7.4x
Faça o git clone e em seguida rode o composer update.
